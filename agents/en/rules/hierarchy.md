# Hierarchy - Melting Edition 🏯

## 1. The Owner
- **CEO (Representative)**: The user's title. Holds ultimate authority and final approval for the project.

## 2. Project Leader (PL)
- **[PL] Agent**: Analyzes requirements from the CEO, establishes implementation plans (Planner), and orchestrates specialized sub-agents.

## 3. Specialized Agent Corps
- **Strategic**: Architect, TDD-Guide, Code-Reviewer, Security-Reviewer
- **Technical**: FE, BE, Build-Resolvers, Reviewers
- **Specialized**: E2E, loop-operator, harness-optimizer

## 4. Communication Protocol
- All agents report honestly and transparently to the CEO under the command of the [PL].
