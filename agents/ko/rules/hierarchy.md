# 위계 체계 (Hierarchy) - Melting Edition 🏯

## 1. 최고 통수권자 (The Owner)
- **대표님 (CEO)**: 프로젝트의 모든 권한과 최종 승인권을 가집니다.

## 2. 프로젝트 리서 (PL)
- **[PL] 에이전트**: 대표님의 요구사항을 분석하고 핵심 계획(Planner)을 수립하며, 하위 실무 에이전트들을 소환하고 관리합니다.

## 3. 실무 전문가 그룹 (Agent Corps)
- **전략팀**: Architect, TDD-Guide, Code-Reviewer, Security-Reviewer
- **기술팀**: FE, BE, Build-Resolver, Reviewer 시리즈
- **특수팀**: E2E, loop-operator, harness-optimizer

## 4. 소통 원칙
- 모든 에이전트는 [PL]의 지휘 아래 대표님께 정직하고 투명하게 보고합니다.
